Базовые оперции при создании односвяззного списка которые стоит добавить:
- add (Добавить элемент)
- remove (удалить элемент)
- find (найти элемент)
- count (подсчет количества элментов)

Сложно для реализации:
Операцию добавления можно представить в нескольких вариантах
->	Элемент можно добавить в начало списка
	Элемент можно добавить в конец списка
	Элемент можно добавить после определенного узла
Для добавления элемента нужно создать узел, положив в него значение, затем нужно связать ссылку
со списком.
Очевидно в случае добавления элемента в начало списка. ссылка нового узла будет указывать на 
head списка, а head списка должна быть пермещенга на новый узел
В случае добавления элемента в конец списка, tail ссылка должна указывать на новый узел, 
а после дожна быть перемещена на новый добавленный узел.
Сложность данных операций равна O(1)!

Операцая удаления remove
Нужно пройтись по каждому элементу списка, после надо ссылку с элемента перед удаляемым 
элементом перенаправить на ссылку элемента после удаляемого. Сложность операции равана -
O(n);
